# vue-boilerplate
Vue boilerplate with Vue-Router, Vuex and andv etc.

## live-demo

[demo](http://donglei.site/vue-boilerplate)

## Set-up

### install dependencies

```bash
npm install
```

### run

```bash
npm run dev
```

### build

```bash
npm run build
```
